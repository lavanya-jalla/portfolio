My Projects

Project 1: Age Calculator

The Age Calculator is a simple web project that takes a user's date of birth as input and calculates their current age in years, months, and days. It is typically built using HTML for structure, CSS for styling, and JavaScript for the core logic. JavaScript handles date operations by comparing the user's birth date with the current date. This project helps beginners understand date manipulation, form handling, and real-time output generation on web pages. It's a practical and easy-to-build project for learning fundamental frontend concepts.

View GitHub Repository

Project 2: TIC-TAE-TOE

This project is a simple implementation of the classic Tic Tac Toe game designed using HTML and CSS. The HTML creates the basic structure of the 3x3 game grid, while the CSS styles the board, cells, and player marks (X and O). Though HTML and CSS handle the layout and appearance, basic JavaScript is usually added to manage game logic, such as detecting wins, switching turns, and resetting the game. It's a great beginner-friendly project to understand web page structure, responsive styling, and event handling.

View GitHub Repository
